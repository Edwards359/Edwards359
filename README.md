## AI Prompt Engineer
**AI‑ассистенты | RAG‑системы | Автоматизация работы с информацией**

Я делаю **практичные AI‑решения для бизнеса**: ассистенты, которые находят ответы в документах, помогают сотрудникам и клиентам, и **снимают рутину** через интеграции и автоматизацию.

---

## Специализация
- **Prompt Engineering**: проектирование промптов под задачи и ограничения (качество, стиль, безопасность, стоимость).
- **AI Assistants**: помощники для поддержки, продаж, операций, внутренних команд.
- **RAG Systems**: ответы “по вашим документам” (база знаний, регламенты, инструкции, FAQ).
- **Workflow Automation**: автоматизация процессов с AI (webhooks, боты, сценарии).
- **LLM Integrations**: встраивание LLM в приложения и сервисы через API.

---

## Что я создаю (What I build)
- **Knowledge assistants** для внутренних баз знаний (поиск + ответ с ссылками на источники).
- **Document search / Q&A** по PDF/Doc/страницам/FAQ (RAG‑пайплайн).
- **AI‑интеграции** в веб‑приложения (API, авторизация, роли, лимиты, логирование).
- **Telegram‑ботов‑ассистентов** для быстрых сценариев (поддержка, заявки, консультации).
- **Автоматизации** (например, n8n): “получил документ → извлёк смысл → ответил → сохранил результат”.

---

## Ценность для клиента (Client value)
- **Экономия времени**: меньше ручного поиска по документам и переписки.
- **Единые ответы**: ассистент отвечает в одном стиле и по актуальным правилам.
- **Контроль качества**: ответы с контекстом/источниками, кеширование, метрики.
- **Быстрый запуск**: MVP за короткий срок → затем улучшения и масштабирование.

## RAG как отдельное направление
**RAG‑системы** — это AI‑решения, которые **сначала находят нужную информацию в документах**, а затем дают **контекстный ответ**, опираясь на найденные фрагменты.

**Примеры применения**
- knowledge assistants для команд
- поиск по документам и регламентам
- корпоративные базы знаний
- FAQ‑ассистенты для поддержки
