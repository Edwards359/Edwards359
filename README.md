## 🧩 RAG как отдельное направление

**RAG‑системы**  
> AI, который сначала ищет информацию в ваших документах, а потом отвечает, опираясь на найденный контент.

🧾 **Use‑cases:**

- внутренние **knowledge assistants**  
- поиск по базе документов / регламентам  
- **FAQ‑ассистенты** для поддержки  
- корпоративные **базы знаний** с понятным интерфейсом
